# GitHub Workflow Practice

This repository is a practice project for setting up GitHub Actions workflows.

## Workflow

This project includes a GitHub Actions workflow that:

1. Runs on every push to the main branch and on pull requests
2. Checks the codebase structure
3. Executes basic tests and linting

## Development

To work on this project:

1. Clone the repository
2. Make your changes
3. Submit a pull request
